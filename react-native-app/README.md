# React Native App 版本

这是一个 Expo 风格的 React Native 最小 App。

## 运行

```bash
cd react-native-app
npm install
npm run start
```

然后可以用 Expo Go 手机 App 扫码，或者运行：

```bash
npm run android
npm run ios
npm run web
```

## 说明

- 使用 `FlatList` 渲染地点
- 使用 `TextInput` 搜索关键词
- 使用横向城市按钮做城市筛选
- 使用 `AsyncStorage` 保存收藏
